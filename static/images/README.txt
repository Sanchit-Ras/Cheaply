# Image Files

This directory contains the following images:

1. `logo.PNG` - The logo for the GroceryCompare website
2. `grocery-bg.svg` - SVG background image for the hero section
3. `grocery-shopping.svg` - SVG image for the about section

These images are used throughout the website. The SVG images are used as placeholders and can be replaced with actual images if desired. If you prefer to use raster images (JPG, PNG), you can replace these SVG files and update the references in the CSS and HTML files accordingly.
